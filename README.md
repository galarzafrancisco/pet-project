# Pet Project

Monorepo to learn and experiment with application development.

## Overview

This repository is a sandbox for learning and experimenting with application development. It is structured as a monorepo and includes both backend and frontend applications, as well as shared packages for types and utilities. The goal is to explore best practices, patterns, and techniques for building applications.
